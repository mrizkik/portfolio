About

Nama saya MUHAMMAD RIZKI, saya adalah anak tertua dari 3 bersaudara. , ketika saya lulus dari SMK, saya benar-benar ingin melanjutkan pendidikan saya ke perguruan tinggi, namun situasi ekonomi keluarga saya tidak memungkinkan saya untuk membayar biaya kuliah yang sangat besar untuk ukuran ekonomi keluarga kami. Kemudian saya bekerja selama satu tahun untuk mengumpulkan uang untuk membayar biaya masuk perguruan tinggi. Saya bekerja di restoran sebagai petugas kebersihan. Setiap mendapatkan gaji saya menyisihkan sebagian uang untuk cita cita saya melanjutkan pendidikan. Saya juga orang yang sangat berdedikasi, untuk memenuhi kebutuhan sehari-hari dan biaya pergi ke kampus. Saya menjual kue dan makanan ringan di balkon, setiap pagi saya mengambil makanan yang akan saya jual di kampus dari tetangga saya. Saya menyimpan barang-barang saya meskipun tidak ada mata kuliah pada waktu itu. tetapi dibalik itu semua ada orang tua yang selalu mendukung dan memberikan segalanya, yang sangat berarti bagi hidup saya.

Saya lulus perguruan tinggi pada Februari, 2018 dengan IPK 2.12. Saya merupakan alumnus salah satu kampus di kota Bandung yaitu, UNIVERSITAS TELKOM. di kampus tersebut saya mengambil program studi TEKNIK KOMPUTER. Program studi terssebut mempelajari tentang Pengelolaan dan implementasi jaringan komputer, Pembangunan sistem komputer yang disesuaikan dengan kebutuhan pengguna, Rekayasa perangkat lunak berorientasi jaringan komputer, Pembangunan sistem kendali dengan komputer. Namun untuk menunjang bidang dan menambah pengetahuan saya juga mengikuti Bootcamp di lembaga pendidikan swasta WEBHOZZ pada penjurusan Front-End Developer, disana saya belajar tentang membangun dan mendesain sebuah website sehingga saya dapat membangun website portofolio seperti yang sedang anda buka.
